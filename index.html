Box Model

Each element on the web page is by default rendered as a box with set of properties. The set of properties that define parts of an element is box model.
Box-sizing is a property that sets how the total width and height of an element is calculated. It has one of two values: content-box (default) or border-box.
Before specifying how width and height properties differ in content-box and border-box cases let's get familiar with padding, border and margin.

Property Meaning Value Example
Padding The spacing between content of the element and its border. 10px
Border The border of an element. Sets vallues for border-width, border-style, and border-color. 10px solid black
Margin The spacing between border of an element and another element. 20px 40px
(*) value of margin and padding can be set separetly for each side: top, right, bottom, left. Syntax consists of one, two or four vallues. One vallue is for all sides. When using two vallues - first is for top and bottom, and second is for right and left side. When using four vallues - one for each side in respacive order: top, right, bottom, left.

When using content-box we set width and height for content only. The width of any border and padding will be added to the final rendered width of an element.
Border-box tells the browser to account for any border and padding in the values you specify for an element's width and height. So value putted for width and height properties will be actuall width and height of rendered element.

Example:

.element1 {
box-sizing: content-box;
width: 400px;
height: 600px;
border: 2px solid black;
padding: 4px;
margin: 10px 20px;
}

The total rendered height of the element1 will be 600(content height) + 4(top padding) + 4(bottom padding) + 2*2(top and bottom border) = 612 (px).

.element2 {
box-sizing: border-box;
width: 400px;
height: 600px;
border: 2px solid black;
padding: 4px;
margin: 10px 20px;
}

The total rendered height of the element1 will be 600px. Content height is calculated: 600 - 4*2(top and bottom padding) - 2*2(top and bottom border) = 588 (px).
